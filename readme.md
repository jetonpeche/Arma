# Docker

Installer docker: `https://www.docker.com/get-started/`  
**Attention** il faut avoir activé la virtualisation dans le BIOS

# config nginx

pour du `localhost`:
- Ouvrer en mode admin `C:\Windows\System32\drivers\etc\hosts`
- Ajouter `127.0.0.1 jetonpeche.fr` (ou un autre nom de domaine) à la fin du fichier

# Allumer l'application

Dans un terminal
- Dans le dossier `/back` faire un `docker compose up`
